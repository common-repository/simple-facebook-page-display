=== Description ===

Simple Facebook Page Display Plugin Easy to Use.
Use Shortcode - [sfpd_page page_url="Your_Facebook_Page_url"]




=== Installation ===
1. Upload `simple-facebook-page-display` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use Shortcode - [sfpd_page page_url="Your_Facebook_Page_url"]
